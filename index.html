Recipes

My intent is to collect recipies for one or two people. Most recipes are from four to six. I will test and comment on them as I go.

Adding Recipes

Make a template file

In the terminal, enter the following:

  1. cd /data/GitHub/odin-recipies/Recipes
  2. cp /data/GitHub/odin-recipies/.template/recipe.template.html /data/GitHub/odin-recipies/Recipes/template.html
  3. ls /data/GitHub/odin-recipies/Recipes/template.html

If the return is the same as the input on the last step, success! We now change the name of the file template.html to the name of your recipe.

Rename the template.html file

$ mv template.html Recipe_Name.html

Edit link in index.html

Edit the file ../index.html so the link in the Recipes list points to the new file

Edit the new recipe file and be sure the 'return to index' link works.
