Recipes

This recipe collection is for recipes that I have found interesting enough to add to it. I'm mostly interested in recipes that are for one or two people and that are easily made in or adapted to air frying or baking in my air-fryer/convection oven. I don't need to risk hot fat burns from deep frying.

I'll try to keep notes on the recipes that I make and I'll try to update recipes as I go. See the notes on the recipes to see what I've rememered to, or found time to update.

This is definitely a work in progress. I'm keeping it under a revision control system (git on GitHub) so if I mess up anything, I'll try to revert and get back what works. Or at least have a fighting chance to do so.

Adding Recipes

This section is for my future self, so I don't have to refigure out what hooks to what. Here's the steps I use, more or less as I learn more.

  1. Copy the recipe template file to the recipes directory.

    The recipe template file is in the file Recipes/recipes/resources/templates/recipe.template.html The file 'index.html' is the root file of this document. The recipe file itself is in the Recipes/recipe directory.

  2. Give the File a Descriptive Name

    Name your new file whatever name you want to access it from the main document. I prefer a descriptive name, because I browse directories and like to know what I'm looking at.
